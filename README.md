# alx-fe-reactjs

This repository contains the ALX Frontend React.js project. It includes a React application scaffolded for ALX tasks and exercises.

## Getting started

1. Install dependencies

```
npm install
```

2. Start the dev server

```
npm start
```

## Build

```
npm run build
```
